# ☁️ 词云生成器 Pro — Word Cloud Generator

一个基于浏览器的智能中文词云可视化工具，支持 AI 辅助文本分析、实时渲染、多格式导出与丰富的自定义配置。

---

## ✨ 功能特性

| 特性 | 说明 |
|------|------|
| 📝 **多源文本输入** | 支持直接粘贴、拖拽上传 `.txt` / `.md` / `.csv` 文件 |
| 🤖 **AI 智能分析** | 接入 DeepSeek API，自动识别停用词、完整词与近义词合并组 |
| 🎨 **丰富的配色** | 内置 6 套专业配色（Viridis / Plasma / Cool / Warm / Ocean / 随机彩色） |
| 📐 **灵活画布** | 4 种预设比例（4:3、3:2、16:9、1:1），支持自定义宽高 |
| ⚙️ **精细控制** | 旋转概率、停用词表、完整词保护、合并词组等高级参数 |
| 📊 **词频统计** | 实时展示总词数、不重复词数、Top 30 词频排行 |
| 💾 **多格式导出** | 一键下载 PNG 图片、JSON 词频数据、CSV 表格 |
| ⌨️ **快捷键支持** | `Ctrl + Enter` 生成、`Ctrl + S` 下载图片 |
| 💡 **状态持久化** | 自动保存所有设置到 `localStorage`，刷新不丢失 |
| 📱 **响应式布局** | 桌面端双栏 + 移动端自适应 |

## 🚀 快速开始

无需安装，直接在浏览器中打开 `index.html` 即可使用：

```bash
# 方式一：直接打开
open index.html

# 方式二：本地服务器（推荐，避免 CORS 限制）
npx serve .
# 或
python -m http.server 8080
```

> **注意**：AI 分析功能需要自备 [DeepSeek API Key](https://platform.deepseek.com/)。

## 🛠️ 技术栈

- **词云渲染**：[wordcloud2.js](https://github.com/timdream/wordcloud2.js)
- **中文分词**：`Intl.Segmenter`（现代浏览器原生 API）+ 降级兼容方案
- **样式方案**：CSS Variables + Flex 布局，无外部 UI 框架依赖
- **数据持久化**：`localStorage`
- **AI 接口**：DeepSeek Chat Completions API（`deepseek-v4-flash`）

## 📖 使用指南

### 1. 输入文本
- 在左侧文本框粘贴中文内容
- 或拖拽 `.txt` / `.md` / `.csv` 文件到上传区域

### 2. 配置参数

| 参数 | 作用 |
|------|------|
| **画布尺寸** | 控制输出图片的像素宽高 |
| **旋转概率** | 0% ~ 100%，调节词语排列方向 |
| **配色方案** | 选择整体色彩风格 |
| **背景颜色** | 白 / 黑 / 灰 / 透明 / 自定义 HEX |
| **停用词** | 过滤无意义虚词，默认含 85 个常见中文停用词 |
| **完整词保护** | 指定“人工智能”“机器学习”等不可拆分的专有名词 |
| **合并词组** | 将“AI / 人工智能 / 人工智慧”合并统计为同一词 |

### 3. AI 辅助（可选）
1. 在「AI 智能分析」区填入 DeepSeek API Key
2. 点击 **🔍 AI 分析文本**
3. 在弹窗中勾选建议的合并词组、停用词、完整词
4. 点击「采纳选中结果」，系统自动回填到对应设置项
5. 再次点击「生成词云」即可获得优化后的可视化效果

### 4. 生成与导出
- 点击 **✨ 生成词云** 或按 `Ctrl + Enter`
- 鼠标悬停词语查看频次，点击词语复制到剪贴板
- 使用悬浮按钮或底部操作栏导出：
  - 📥 **PNG 图片**（带透明/背景色）
  - 📋 **JSON 词频数据**
  - 📊 **CSV 词频表格**

## 🔒 隐私说明

- 所有文本处理均在**本地浏览器**完成，不会上传至任何服务器
- API Key 与设置仅保存在浏览器 `localStorage` 中
- 如需清除敏感信息，点击「清除 API Key」或清空浏览器本地存储
